# BeaTUI

BeaTUI is a small terminal-based beat maker written in Go.

It lets you create simple beats using a 16-step sequencer, tweak a basic kick instrument, and export the result as a WAV file â€” all directly from the terminal.


---

## Installation & Run

Requirements:
- Go 1.21+

```bash
go mod init beatui
go mod tidy
go run .
```
